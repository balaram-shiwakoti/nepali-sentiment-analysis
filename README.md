# 🧠 भावना-बुझ (Bhavana-Bujh): Nepali Sentiment Classifier

Bhavana-Bujh is a lightweight Natural Language Processing (NLP) application designed to analyze and classify the sentiment of Nepali text into **Positive (सकारात्मक)**, **Negative (नकारात्मक)**, or **Neutral (तटस्थ)** categories.

## 🛠️ प्रविधिहरू (Technologies Used)
* **Python 3.8+**
* **Scikit-Learn** (TF-IDF Vectorizer & Multinomial Naive Bayes)
* **Streamlit** (For the interactive Web UI)
* **Pandas** (For dataset management)

## 🏃‍♂️ कसरी रन गर्ने? (How to Run)

1. **Dependencies इन्स्टल गर्नुहोस्:**
   ```bash
   pip install streamlit scikit-learn pandas

```

2. **मोडल ट्रेनिङ गर्नुहोस्:**
```bash
python train_nlp.py

```


3. **वेब एप सुरु गर्नुहोस्:**
```bash
streamlit run app.py

```
